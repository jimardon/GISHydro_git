GISHydroWEB

GISHydroWEB is a GIS-based web application for performing hydrologic analysis developed by The University of Maryland in conjunction with Maryland State Highway Administration. It consists of a database of hydrologic layers and customized modeling tools.

Some Features Include:

  • Complete database of DEM, Land use, and Soils data for Maryland drainage areas (includes parts of DE, PA, VA, and WV).
  • Watershed delineation.
  • Watershed statistics (drainage area, CN, Tc, and more).
  • Peak discharge estimates from USGS regional regression equations (with confidence intervals and gauge calibration).
  • Built-in WinTR-20 model.

To start go to the "Area of Interest" tab

GISHydroWEB is under continuous revision. Although we are not a web application development company and do not charge for the use of our website, we do try to support it to the best of our ability. The application has a Help tab that contains some useful documentation.

Check more on the GISHydro website.

Area of Interest

To start the application draw the extent of the area of interest containing the potential watershed. Select the available options and then click Apply. It could take a couple of minutes to process depending on the size of the AOI. Zoom in to enable stream and road layers.

Subwatershed

Subdivide the watershed into subwatersheds to create stream reaches that will define the stream network topology for WinTR-20 modeling (optional). After clicking Yes the option to add flow paths or outlets will become available. Otherwise, the Time of Concentration tab will be enabled, this could take a couple of seconds while GISHydroWEB gathers all the required information to continue the analysis.

Subdivide

Flow paths and outlets will divide the watershed in subwatersheds. At least one flow path must be selected. Outlets must be placed on valid streams. After this process has finished you can download the subwatersheds in GeoJSON format or proceed to the Download tab to e-mail it as a package.

Time of Concentration

Select a time of concentration method and options, then click Compute Tc. This could take some time depending on the amount and size of the subwatershed(s). Tapping on a subwatershed will show up the respective ID# (available after Tc estimate). If the Velocity Method is selected, clicking on Show Tc will allow the user to merge by segment and recompute the Tc values in a new window. Otherwise, the Show Tc button will only show the Tc values for each subwatershed.

WinTR-20

To create the WinTR-20 input, error, and output files first some events must be selected. This could take some time depending on how many events are selected. After this process has finished some options will become available, press the Create WinTR-20 button to create input, error, and ouput files to download. Please check WinTR-20's USDA website for more information on WinTR-20.
